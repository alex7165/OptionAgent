# OptionAgent

## Ziel

OptionAgent ist ein persönlicher KI-Agent zur Analyse von Optionsstrategien.

Der Agent unterstützt bei der Analyse von:

- Earnings Trades
- Short Strangles
- Short Straddles
- Covered Calls
- Cash Secured Puts
- weiteren Optionsstrategien

Der Agent führt niemals selbstständig Orders aus.
